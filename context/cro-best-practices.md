# CRO Best Practices - Zero Vape

## Above-the-Fold (Category/Product)

### 5-Second Clarity
المستخدم لازم يفهم بسرعة:
1. وش المنتج/الفئة
2. هل يناسبه
3. السعر/القيمة
4. الخطوة التالية (CTA)

### Required Elements
- عنوان واضح فيه الكلمة الشرائية.
- سطر قيمة مختصر (ميزة + فائدة).
- CTA رئيسي: "أضف للسلة" أو "اطلب الآن".
- إشارة ثقة: شحن/استرجاع/دفع مرن.

## CTA Strategy

### Primary CTA
- "أضف للسلة"
- "اطلب الآن"

### Secondary CTA
- "شاهد الأكثر مبيعًا"
- "تصفح العروض"

### CTA Placement
1. أعلى الصفحة
2. بعد المقارنة/الفوائد
3. نهاية الصفحة

## Trust Signals
- روابط واضحة: الشحن، الاستبدال، FAQ.
- عرض خيارات الدفع (بما فيها التقسيط عند توفره).
- إبراز الأكثر مبيعًا.

## Objection Handling

### Objection: "مو عارف ايش أختار"
- **Response**: قدّم مقارنة قصيرة + رابط الأكثر مبيعًا.

### Objection: "خايف ما يناسبني"
- **Response**: اربط صفحة الأسئلة الشائعة + التوافقات.

### Objection: "السعر عالي"
- **Response**: حوّل إلى البكجات والعروض + القيمة مقابل الاستخدام.

## Form/Checkout Principles
- تقليل خطوات الإتمام.
- توضيح تكلفة الشحن مبكرًا.
- عدم إجبار العميل على إنشاء حساب قبل الشراء إذا أمكن.

## Mobile CRO
- أزرار كبيرة واضحة.
- CTA ثابت أو مكرر على الصفحة الطويلة.
- ترتيب المحتوى: الفائدة -> السعر -> CTA -> الثقة.

## Content-to-Commerce Flow
- كل مقال لازم يدفع لصفحة تصنيف أو منتج مباشرة.
- داخل المقال:
  - CTA أول في الثلث الأول
  - CTA ثانٍ بعد الجزء العملي
  - CTA أخير في الخاتمة
